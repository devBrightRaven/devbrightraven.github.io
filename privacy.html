Who we are

brightraven.world (this website)

This website does not use cookies, analytics, or tracking scripts. No personal data is collected through this website. Hosting is provided by GitHub Pages; their infrastructure may log IP addresses in server logs per GitHub's privacy statement.

Maida (desktop application)

Maida is a desktop application for Windows and Linux. All game library data, behavior logs, and preferences are stored locally on your device and never leave it.

Data stored on your device

  • Game library data (read from local Steam installation files)
  • Session logs (your TRY / NOT NOW choices, auto-pruned after 30 days)
  • Kata groupings and preferences
  • Theme, language, and UI settings

This data is stored in your app data directory and is never transmitted.

Telemetry (opt-out)

By default, Maida sends a single anonymous ping each time the app launches:

  • Anonymous ID — a randomly generated UUID, not linked to any account
  • Days since first launch
  • App version

This ping contains no game data, no behavior data, no device information, and no personal identifiers. It is sent to a Cloudflare Worker endpoint and stored in Cloudflare D1 (APAC region). Cloudflare infrastructure logs may include your IP address.

You can disable telemetry at any time in Settings. When disabled, no network requests are made except for update checks and image loading.

Auto-updater

Maida checks for updates via GitHub Releases on launch (GET request only). No personal data is sent.

Game cover images

Cover images are loaded from Steam CDN (Akamai) and optionally IGDB. These are standard HTTP image requests.

IGDB integration (optional)

If you manually enter Twitch Developer credentials, Maida queries the IGDB API for game duration data. Credentials are stored in your operating system's secure keyring (Windows Credential Manager / Linux libsecret), not in plain text files. You can clear them in Settings.

What we do not do

  • We do not create user accounts
  • We do not collect names, email addresses, or phone numbers
  • We do not track behavior patterns across sessions
  • We do not share data with third parties (beyond Cloudflare infrastructure)
  • We do not serve advertisements
  • We do not use cookies or analytics services
  • We do not read Steam account passwords or login credentials
